Oracle

  1. dual表->虚拟表，可以用来测试语法规则

  2. 记得安装最后，口令管理接触scott账号锁定，并设置新密码

  3. cmd sqlplus / as sysdba 以管理员身份登录

  4. 创建用户 create user student identified by root;

  5. 说明：indentified by 后面是密码

    修改用户密码：

    ​ alter user user01 identified by user10;

  6. connect by 用于存在父子，祖孙，上下级等层级关系的数据表进行层级查询。

  7. 每个session是独立的，当前session没有commit，其他session无法获取数据

  8. sqlplus user\passwd 登录

  9. row_number的用途非常广泛，排序最好用它，它会为查询出来的每一行记录生成一个序号，依次排序且不会重复，注意使用row_number函数时必须要用over子句选择对某一列进行排序才能生成序号。

  10. rank()代表级别，over 指定排序规则， 会跳号，即记录同一排名的人数

  11. dense_rank() 与rank()类似，但不会记录同一排名的人数，即排名是按序的

  12. partition by 分组在进行排序，eg: partition by km ORDER BY fs DESC

  13. null不等于任何值，所以任何与null相关的比较包括>,<,=都不成立

    对于null只可以用 is null或者is not null来判断

  14. nvl() 若第一个参数为null，则值为第二个参数；

  15. rownum指从查询返回的行的编号

  16. 日期默认格式是 DD-MON-YYYY 查询条件按照日期来，日期也要加单引号

  17. 会话级临时表采用 on commit preserve rows ; 而事务级则采用 on commit delete rows ;

  18. drop分区会导致全局索引失效

  1. execute immediate 后面执行的DML语句可以是还未正式创建的表，是在存储过程中创建的表（可以使用using动态传参，参数占位符： eg: :1, :2, :3

3.程序主体
变量的赋值
变量赋值一般通过 ：=的方式
比如
declare
i number;
j varchar(10);
m date;
begin 
  i:=100;
  j:='test';
  m:=sysdate;
  dbms_output.put_line('i='||i);
  dbms_output.put_line('j='||j);
  dbms_output.put_line('m='||m);
end;

也可以通过游标fetch的方式赋值，这个在讲到游标的方式

支持的表达式
数字表达式： 
+ 加
- 减
* 乘
/ 除
字符表达式 
|| 拼接2个字符和concat一样
关系表达式    
< 小于     
> 大于    
= 等于（不是赋值运算符:=）     
like 类似于     in 在……之中

>= 大于等于     
!= 不等于 或<>     
between 在……之间     
注意：关系型表达式运算符两边的表达式的数据类型必须一致。     
逻辑表达式    
NOT：逻辑非     
OR：逻辑或     
AND：逻辑与     
注意：运算的优先次序为NOT、AND和OR。

  3. Oracle COALESCE函数语法为COALESCE(表达式1,表达式2,…,表达式n)，n>=2,此表达式的功能为返回第一个不为空的表达式，如果都为空则返回空值。

    注意：所有表达式必须为同一类型或者能转换成同一类型

  4. q(Q)‘表达式，会保证引号内的字符串格式不变，防止因为’，”， 特殊字符等造成字符串的混乱。
